# S-Py-re

A bare-bones deckbuilder roguelike inspired by Slay the Spire, built with Python and Pygame.

## Purpose

Learning project to practice game development and implement card game mechanics.

## Requirements

- Python 3.12+
- Pygame

## Installation
```bash
pip install pygame
```

## Usage
```bash
python main.py
```
